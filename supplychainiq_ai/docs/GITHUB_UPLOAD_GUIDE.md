# GitHub Web Upload Guide

1. Extract the ZIP file.
2. Create a new empty GitHub repository.
3. Select **Add file → Upload files**.
4. Upload the contents inside the extracted project folder.
5. Upload in batches if necessary:
   - Root files
   - app
   - src
   - scripts
   - tests
   - docs
   - data
6. Commit each batch.
7. Do not upload `.venv`, `__pycache__`, or private credentials.
